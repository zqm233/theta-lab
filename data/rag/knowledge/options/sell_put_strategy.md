# Sell Put 策略完全指南

## 什么是 Sell Put？

Sell Put（卖出看跌期权）是一种**做多**策略，通过卖出看跌期权收取权利金，同时承担在行权价买入股票的义务。

## 核心机制

### 收益结构

- **最大收益**：权利金（Premium）
- **盈亏平衡点**：行权价 - 权利金
- **最大亏损**：(行权价 - 权利金) × 100（理论上行权价可跌至 0）

### 四种结局

1. **股价 > 行权价**（价外，OTM）：
   - 期权作废，卖方保留全部权利金
   - **最理想结果**

2. **股价略低于行权价但 > 盈亏平衡点**：
   - 被行权，以行权价买入股票
   - 持有股票浮亏 < 权利金，仍有净利润

3. **股价 = 盈亏平衡点**：
   - 被行权买入
   - 持有股票浮亏刚好等于权利金，不赚不亏

4. **股价 < 盈亏平衡点**：
   - 被行权，以高于市价买入
   - 持有股票浮亏 > 权利金，净亏损

## 关键指标

### 1. 安全垫（Safety Cushion）

**定义**：行权价与当前股价的距离

```
安全垫% = (行权价 - 当前价) / 当前价 × 100%
```

**选择建议**：
- **保守**：安全垫 > 15%（深度价外）
- **平衡**：安全垫 10-15%（略价外）
- **激进**：安全垫 5-10%（接近价外）
- **极激进**：安全垫 < 5%（接近平值或价内）

### 2. ROIC（Return on Investment Capital）

**定义**：权利金占保证金的百分比

```
ROIC = 权利金 / (行权价 × 100) × 100%
```

**年化收益率**：

```
年化收益率 = ROIC × (365 / DTE)
```

**选择建议**：
- 目标年化收益率：15-30%
- ROIC 通常 1-5%（取决于 DTE 和安全垫）

### 3. Delta

**定义**：期权价格相对股价的敏感度

- Put 的 Delta 为负值（-0.05 到 -1.0）
- Sell Put 持仓 Delta 为正（等效做多）

**选择建议**：
- **保守**：Delta 0.10-0.20（触及概率 10-20%）
- **平衡**：Delta 0.25-0.35（触及概率 25-35%）
- **激进**：Delta 0.45-0.55（触及概率 45-55%）

### 4. Theta（时间价值衰减）

**定义**：每天期权价值因时间流逝而减少的金额

- Sell Put 的 Theta 为正（时间对你有利）
- 越接近到期，Theta 越大（非线性加速）

**DTE 选择**：
- **30-45 天**：Theta 适中，年化稳定，调整空间大
- **14-21 天**：Theta 加速，短期策略最佳平衡点
- **< 7 天**：Theta 最大但 Gamma 风险激增

## 选择标的的考量

### TSLA vs TSLL

| 标的 | 特点 | 适合人群 |
|------|------|---------|
| **TSLA** | 波动适中，流动性好，财报影响大 | 新手到进阶 |
| **TSLL** | 2倍杠杆 ETF，IV 更高，权利金更丰厚 | 进阶到激进 |

### 选择原则

1. **高 IV（隐含波动率）**：
   - IV Rank > 50% 时卖出最划算
   - 避免 IV 过低时（< 30%）卖出

2. **财报考量**：
   - 避免持仓跨越财报（除非刻意利用 IV Crush）
   - 财报前 1-2 周 IV 飙升是卖出好时机

3. **流动性**：
   - Bid-Ask Spread < 5% 权利金
   - Open Interest > 100

## 风险管理

### 1. 仓位控制

- **单个标的**：不超过总资产 10-15%
- **单次卖出**：1-3 张合约（相当于 100-300 股）
- **总期权敞口**：不超过总资产 30-40%

### 2. 止盈策略

- **50-75% 权利金**：提前平仓锁定利润（Theta 已大部分实现）
- **90% 权利金**：持有到期或最后几天平仓
- **不要贪恋最后 10-20%**：时间价值递减，Gamma 风险增大

### 3. 止损策略

- **浮亏 = 2 × 权利金**：考虑平仓止损
- **股价持续下跌且技术破位**：果断止损
- **被动接股**：如果长期看好标的，可以接受被行权（变成长线持股）

### 4. 展期（Roll）

当行权价接近或跌破当前价时：

**向下展期（Roll Down）**：
- 平仓当前 Put
- 卖出更低行权价的 Put（相同到期日或更远）
- 收取额外权利金，但风险降低

**向后展期（Roll Out）**：
- 平仓当前 Put
- 卖出相同行权价、更远到期日的 Put
- 收取额外权利金和时间

## 实战案例

### 案例 1：TSLA 保守 Sell Put

**市场环境**：
- TSLA 现价：$250
- IV Rank：65%（高波动环境）
- 无近期财报

**策略选择**：
- 行权价：$220（安全垫 12%）
- 到期日：30 天后
- 权利金：$3.50
- Delta：0.25

**分析**：
- ROIC：3.50 / 220 = 1.59%
- 年化收益率：1.59% × (365/30) ≈ 19.3%
- 盈亏平衡点：$216.50
- 风险：股价需跌 13.4% 才会亏损

**结果预测**：
- 概率 75%：到期时 > $220，赚取全部权利金 $350
- 概率 20%：到期时 $216-220，被行权但仍盈利
- 概率 5%：到期时 < $216，亏损

### 案例 2：TSLL 激进 Sell Put（财报前）

**市场环境**：
- TSLL 现价：$12
- TSLA 财报前 3 天
- IV Rank：90%（极高）

**策略选择**：
- 行权价：$11（安全垫 8.3%）
- 到期日：财报后 1 周（10 天）
- 权利金：$0.80
- Delta：0.40

**分析**：
- ROIC：0.80 / 11 = 7.27%
- 年化收益率：7.27% × (365/10) ≈ 265%（极高）
- **风险**：财报利空导致 TSLA 暴跌 > 10%

**实战建议**：
- 仅适合激进交易者
- 准备好财报后立即平仓（利用 IV Crush）
- 或愿意接股持有

## 常见错误

### 1. 追逐高权利金

**错误**：看到权利金 $10 很诱人，不看安全垫就卖

**后果**：行权价可能接近或低于现价（价内 ITM），极易被行权

### 2. 忽视财报

**错误**：持有跨越财报的 Put，以为「财报利好就稳赚」

**后果**：财报不及预期 → 股价暴跌 → 巨额亏损

### 3. 死扛到底

**错误**：被行权后股价继续跌，仍死握股票希望「回本」

**后果**：变成长期套牢，资金机会成本

### 4. 全仓一个到期日

**错误**：5-10 张 Put 全是同一天到期

**后果**：如果当天市场崩盘，所有仓位同时爆炸

## 进阶技巧

### 1. 组合策略

- **Bull Put Spread**（看涨价差）：卖出高行权价 Put，买入低行权价 Put，限制风险
- **Iron Condor**：同时卖出 Put 和 Call，赚取震荡市权利金
- **Wheel 策略**：Sell Put → 被行权 → Sell Covered Call → 循环

### 2. IV Rank 择时

- **IV Rank > 70%**：绝佳卖出时机（恐慌时刻）
- **IV Rank 30-70%**：可以操作
- **IV Rank < 30%**：避免卖出（权利金太薄）

### 3. 技术面配合

- **支撑位附近卖 Put**：股价回调到关键支撑位时，在支撑位下方卖 Put
- **趋势跟随**：上升趋势中卖 Put 成功率更高

### 4. 资金管理

- **预留现金**：保证金的 2-3 倍现金应付紧急情况
- **分批建仓**：不要一次性卖出全部仓位
- **多样化**：分散到期日、分散标的

## 检查清单

**开仓前必查**：

- [ ] IV Rank > 30%（最好 > 50%）
- [ ] 安全垫 > 10%
- [ ] 距离财报 > 2 周
- [ ] Delta 在 0.15-0.40 之间
- [ ] 年化收益率 > 15%
- [ ] Bid-Ask Spread 合理
- [ ] 有足够现金接股或平仓止损

**持仓管理**：

- [ ] 每日检查 Delta 变化
- [ ] 盈利 50-75% 考虑提前平仓
- [ ] 亏损 2× 权利金考虑止损
- [ ] 到期前 1 周评估是否展期
- [ ] 财报前 1-2 天平仓（除非策略就是吃 IV Crush）

## 心理素质

### 成功 Sell Put 的心态

1. **接受被行权**：卖 Put 就是说「我愿意在这个价格买入」，不要被行权后后悔
2. **不追求完美**：赚 50-75% 就走，不要等 100%
3. **止损果断**：错了就认，不要「死扛」变成「股民」
4. **稳定为王**：追求稳定的 20-30% 年化，不要贪 100%+

### 常见心理陷阱

- **赌徒心态**：「这次肯定不会跌」→ 卖太多或太接近现价
- **锚定效应**：「我 $200 卖的，现在 $180 一定要等回 $200」→ 不止损
- **FOMO**：「别人都在赚，我也要卖」→ IV 低时盲目卖出
